# 🏙️ DizZine - City Posters Experience

**DizZine** es una aplicación interactiva de posters minimalistas que celebra la arquitectura y la esencia de las principales ciudades de Colombia. Diseñada con un enfoque en la experiencia de usuario (UX) y transiciones fluidas.

## 🚀 Características Principales

- **✨ Navegación Inteligente:** Implementación de `Intersection Observer` para detectar la ciudad en pantalla y actualizar el Header dinámicamente.
- **📱 Responsive Design:** Adaptado para una experiencia de pantalla completa con `scroll-snap`.
- **⏳ UX Optimizada:** Pantalla de carga (Loader) personalizada para una transición suave de datos.
- **🛠️ Arquitectura Limpia:** Comunicación eficiente entre componentes (Props, Lifting State Up, Hooks).
- **📈 Contador de Interacciones:** Seguimiento local de consultas por ciudad.

## 🛠️ Stack Tecnológico

- **React.js** (Hooks: `useState`, `useEffect`, `useRef`)
- **CSS3** (Variables, Flexbox, Keyframes para animaciones)
- **Vite** (Build Tool ultra rápido)

## 📸 Demo Visual

> "El diseño no es solo lo que se ve, es cómo funciona."

El proyecto utiliza un sistema de **Scroll Suave** y **Estados Dinámicos** para que cada ciudad tenga su propia identidad visual a través de colores de fondo dinámicos.
